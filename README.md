This repository contains basic Java programs for learning and practice.

* Hello World Program
* Calculator Program
* Exception Handling Example
* Multithreading Example

---

## ☕ Example Java Program

```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

### 📌 Description

This program prints **Hello, World!** on the console.
It is the first program most beginners write when learning Java.

---

## 🛠 How to Run

1. Install Java (JDK).
2. Save the file as `HelloWorld.java`.
3. Compile the program:

```
javac HelloWorld.java
```

4. Run the program:

```
java HelloWorld
```
